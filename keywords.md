# Python Basic Keywords (For Beginners)

| 한국어 (Korean) | 영어 (English) | 설명 (Description) |
| :--- | :--- | :--- |
| **함수** | **Function** | 특정 작업을 수행하는 코드의 묶음 (`def` 키워드로 정의) |
| **변수** | **Variable** | 데이터를 저장하는 공간 |
| **리스트** | **List** | 여러 데이터를 순서대로 담는 바구니 (`[]`) |
| **사전 (딕셔너리)** | **Dictionary** | 키(Key)와 값(Value)의 쌍으로 데이터를 저장 (`{}`) |
| **문자열** | **String** | 텍스트 데이터 |
| **정수** | **Integer** | 소수점이 없는 숫자 |
| **모듈/라이브러리** | **Module/Library** | 미리 만들어진 기능들의 집합 (`import`로 불러옴) |
| **반복문** | **Loop** | 코드를 여러 번 반복 실행 (`for`, `while`) |
| **조건문** | **Conditional** | 조건에 따라 다른 코드를 실행 (`if`, `else`) |
| **판다스** | **Pandas** | 파이썬에서 표(Table) 데이터를 다루는 가장 유명한 라이브러리 |
| **데이터프레임** | **DataFrame** | 판다스에서 사용하는 표 형태의 데이터 구조 |
| **결손치 (널 값)** | **Missing Value (NaN/Null)** | 데이터가 비어 있는 상태 |
| **중복** | **Duplicate** | 같은 데이터가 여러 번 나타나는 것 |
| **인덱스** | **Index** | 데이터의 위치나 식별 번호 |
| **정렬** | **Sort** | 특정한 기준에 따라 순서를 바꾸는 것 |
| **예외 처리** | **Exception Handling** | 에러가 발생했을 때 프로그램이 멈추지 않게 처리 (`try`, `except`) |
